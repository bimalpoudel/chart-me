# chart-me

A WordPress Plugin to display page access history in Charts.


## Logic

Keep a record of all page accesses

Page ID | Date     | Counter |
-------:|----------|--------:|
      12|2016-04-25|        6|
      15|2016-04-25|       10|

Use this data for charting purpose.


## Chart APIs (pick one of)

 * AM Chart
 * Fusion Chart
 * Google Charts
